# Prueba Técnica

### Christian Carrillo Zúñiga

#### requerimientos

1. node 16+
2. copiar el archivo `.env.example` a `.env`
3. instalar dependencias
   ```
   npm install
   ``` 
   o

   ```
   yarn
   ```
4. levantar servidor de desarrollo
   ```
   npm run dev
   ```
   o
   
   ```
   yarn dev
   ```
